# zztyyh
SNH48Group机器人<br>
主要功能：
	（1）同步成员口袋房间消息
	（2）同步微博用户动态
	（3）同步摩点项目集资
	（4）发送房间消息、微博、集资到指定QQ
	（5）提供可视化操作配置界面
	（6）其他
开发所属：GNZ48-章泽婷应援会
	虽然人已不在，但项目还是会继续更新和优化，个人兴趣。 <br>
